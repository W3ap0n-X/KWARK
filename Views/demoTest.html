Test #1

This test is to test performance of multiple windows being run by the script.

The NEW button below will open a second Blank window. Once it opens, grab the title bar and move it around your screen. The window should move fluidly or with minimal delayed response.

If the window is lagging, the SET button below can be used to adjust the Window polling rate by changing the numeric value of it's Input. The number in this field represents the number of milliseconds between refresh cycles of the window. Essentially this is the number of milliseconds before the window checks it's variables for updates.

If setting the window refresh rate doesn't change the behavior try using the STOP button below to halt the window polling. You should see the clock at the bottom of this window stop as an indicator that the polling has stopped.
